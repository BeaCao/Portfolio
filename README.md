🚀 **Ver en vivo:** [https://beacao.github.io/Portfolio/](https://beacao.github.io/Portfolio/)

---

## ✨ Características del Proyecto

- **Diseño Ultra-Premium**: Interfaz moderna basada en efectos de cristal (glassmorphism), auroras dinámicas y micro-interacciones suaves.
- **GSAP Animations**: Experiencia de usuario fluida con animaciones de entrada, efectos de scroll y parallax.
- **Mobile First**: Totalmente responsivo y optimizado para una experiencia táctil impecable.
- **Secciones Detalladas**:
  - **Hero**: Presentación dinámica con efecto de escritura.
  - **Proyectos**: Showcase interactivo con enlaces directos a demos reales.
  - **Skills**: Visualización de stack tecnológico mediante bento-grid.
  - **Formación y Contacto**: Información estructurada y accesible.

---

## 🛠️ Stack Tecnológico

- **Frontend**: [Vue 3](https://vuejs.org/) (Composition API + TypeScript)
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Animaciones**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Despliegue**: GitHub Actions & GitHub Pages

---

## 🚀 Despliegue Automático

Este proyecto utiliza **GitHub Actions** para el despliegue continuo. Cada vez que se realiza un commit a la rama `main`, el workflow:
1. Instala las dependencias.
2. Compila el proyecto con Vite.
3. Despliega automáticamente los archivos estáticos en GitHub Pages.

---

## 👩‍💻 Instalación Local

Si deseas clonar este proyecto y ejecutarlo localmente:

```bash
# Clonar el repositorio
git clone https://github.com/BeaCao/Portfolio.git

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```
